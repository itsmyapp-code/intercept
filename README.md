# Intercept

A Progressive Web App (PWA) notification manager for scheduling and holding notifications.

## Features
- Mobile-optimised, scannable dashboard
- Scheduling rules (e.g., "Deep Work", "Out of Hours")
- Application configuration (toggle apps as Allowed/Excluded per schedule)
- Stream view of held notifications
- Offline-first with service worker and manifest
- Dark mode by default
- All UI text uses UK English spelling

## Tech Stack
- Next.js (App Router)
- Tailwind CSS
- Lucide React Icons
- IndexedDB/localStorage (future)

## Hosting
- GitHub (source control)
- Vercel (deployment)

## Getting Started
```bash
npm install
npm run dev
```

## PWA
- Manifest: `public/manifest.json`
- Service Worker: `public/service-worker.js`
