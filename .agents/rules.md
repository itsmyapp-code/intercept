# Agent Rules and Guardrails

## Tech Stack
* **Frontend**: Next.js (App Router)
* **Styling**: Tailwind CSS
* **Icons**: Lucide Icons (`lucide-react`)
* **Android Native Layer**: Kotlin

## Language and Spelling
* **Exclusively UK English** spelling must be used across the entire application interface, logs, and variables.
* Examples:
  * `customisation` instead of `customization`
  * `prioritised` instead of `prioritized`
  * `processing`
  * `responses`
  * `optimised` instead of `optimized`
  * `cancellation` instead of `cancelation`
  * `cancelled` instead of `canceled`
  * `programme` instead of `program` (except where referring to computer programs/code context if required, but UI/logs/variables must use UK spelling)
  * `colour` instead of `color`

## Theme
* **Theme**: Ultra-minimalist, high-contrast dark theme.
* **Target Layout**: Mobile/PWA layouts.
* **Palette**: Charcoal backgrounds, pure white text, bold borders, minimalist accent colors (like pure orange `#FF5722` or red/green indicators).
